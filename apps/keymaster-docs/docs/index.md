## Feature Overview

<div class="km-feature-grid">
  <div class="km-feature-card">
    <h3>React / Vue Shortcut Libraries</h3>
    <p>
      <code>@keekuun/keymaster-react</code> for React and <code>@keekuun/keymaster-vue</code> for
      Vue provide Hook / Composition API wrappers, focusing on business logic rather than event
      details.
    </p>
  </div>
  <div class="km-feature-card">
    <h3>Interactive Demos</h3>
    <p>
      Built-in interactive demos (supporting shortcuts like <kbd>Ctrl</kbd>+<kbd>S</kbd>) allow you
      to experience the library's behavior without setting up a project.
    </p>
  </div>
  <div class="km-feature-card">
    <h3>Best Practices & Design Guidelines</h3>
    <p>
      Design recommendations for editor, list operations, and other scenarios to help you create
      powerful yet unobtrusive keyboard interactions.
    </p>
  </div>
  <div class="km-feature-card">
    <h3>Core Module Architecture</h3>
    <p>
      <code>@keekuun/keymaster-core</code> provides shared type definitions and parsers, ensuring
      type consistency and code reuse across React and Vue versions.
    </p>
  </div>
  <div class="km-feature-card">
    <h3>Search & Source Links</h3>
    <p>
      Built-in local search and GitHub links in the top-right corner for quick API lookup, source
      code browsing, and contributions.
    </p>
  </div>
  <div class="km-feature-card">
    <h3>Multi-Version Management</h3>
    <p>
      The documentation site always displays the latest version. If you're using an older version,
      check the <a href="/versions">version management documentation</a> to learn how to view
      documentation and APIs for specific versions.
    </p>
  </div>
</div>
